📄 Formula Sheets

Math formulas in one place.

Quick formulas for algebra, geometry, algebra 2, and calculus.

✏️ Algebra 1

Slope: m = (y₂ - y₁) / (x₂ - x₁)

Slope-intercept: y = mx + b

Point-slope: y - y₁ = m(x - x₁)

Percent change: (new - old) / old × 100

📈 Algebra 2

Quadratic formula: x = (-b ± √(b² - 4ac)) / 2a

Exponent rule: aᵐ · aⁿ = aᵐ⁺ⁿ

Log rule: log(ab) = log(a) + log(b)

Arithmetic sequence: aₙ = a₁ + (n - 1)d

📏 Geometry

Triangle area: A = 1/2bh

Circle area: A = πr²

Circumference: C = 2πr

Pythagorean theorem: a² + b² = c²

📊 Calculus

Power rule: d/dx(xⁿ) = nxⁿ⁻¹

Constant rule: d/dx(c) = 0

Sum rule: d/dx(f + g) = f' + g'

Basic integral: ∫xⁿ dx = xⁿ⁺¹/(n+1) + C
